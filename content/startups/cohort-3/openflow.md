Openflow (formerly Opentest) scans a codebase, discovers every API endpoint, and exposes them as MCP tools that AI agents can call directly inside developer environments like Cursor, Claude Code, Windsurf, and Cline. It wraps each endpoint as a typed tool with validation and auth pre-wired.

The Flow product simulates real user flows to catch broken releases before they ship — not by running tests, but by simulating actual users. Founded by Harvey Mudd College students Alspencer Omondi and Ethan Sandoval, who won the 2024 HMC Dean Sundberg Prize.
